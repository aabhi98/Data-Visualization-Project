Retrospective Analysis of Events at Abila City Using Visualization Techniques.

Description:

In this project, we are trying to answer the three questions posed in the VAST 2021 Mini Challenge 3 by using the call center data and the microblog data provided. The dataset contains tweets/messages about the three major events happened at Abila City and also some unrelated spam/chatter information. The beeswarm graph, word cloud, pie chart, bar chart and arc diagram helps the user to analyze the data and answer the first task of the challenge to characterize the data into different events happening at Abila. The spline graph helps us understand the risk/threat to public at a given point in time by answering the second task of the challenge. Point Map helps to visualize the events at Abila by visually representing the locations of the messages tweeted. Bigger circles on the point map represents where the three major events happened and helps us answer the third task of the challenge which is to help the user understand where to send the first responders if there was a chance.

Interactions:

Beeswarm graph has a time slider which can be slided to left or right and this changes the word clouds, pie charts and bar charts. The check boxes on top of the page contains the three major events happening at Abila. When a checkbox is unselected/selected, the changes will be reflected in word clouds, pie charts and bar charts. Word cloud represents the most tweeted words in that time frame. Beeswarm also has a tooltip that shows the author and message tweeted. Bar chart has dropdown to show most tweeted tags or authors during that time frame. Pie chart shows the distibution of the sentiment Analysis of the tweets in that time frame with a tooltip. Arc diagram, when hovered on a circle, connects all the authors that tweeted about the same major event that author tweeted. Point Map shows animation of the location of the tweets categorized via event types when the play button is clicked. The range Slider is used to show animation of the data points from that time to the end of the time frame. The "Track Van" button is used to show the "hit and run" van path over time. When hovered on a circle, lines will be connected to the circles with the same type of event. The circle also shows a tooltip with the data related to that message.
